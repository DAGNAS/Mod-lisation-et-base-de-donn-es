# Modélisation-et-base-de-données
Projet A1 : Nous faisons partie de l’équipe « Data » de l’entreprise Clear Data. Nous sommes  chargées de créer une base de données non redondante et cohérente comportant des  relevés publics sur la qualité de l’air dans les grandes villes de France. Cette liste pourra  être modifiée, consultée, ajoutée et supprimée facilement.
